Java Programming

STEP 1

BASICS

This part of the tutorial covers the basics of the language, including: variables, operators, expressions, statements, blocks and control flow statements.

STEP 2

LOOPS ,METHODS

Java Loops are programming structures that help a certain block of code to be executed repeatedly until a certain defined condition is met.

STEP 3

ARRAYS ,STRINGS

An array is a linear data structure that holds a group of elements having the same data types. A string is an object that defines a series of characters.

STEP 4

OOP

OOP is faster and easier to execute. OOP helps to keep the Java code DRY "Don't Repeat Yourself", and makes the code easier to maintain, modify and debug

ADVANCED TOPICS

Advanced Java covers some topics like web services, database connectivity, JSP, Servlets, EJB,

STEP 6

PACKAGES

Package in Java is a mechanism to encapsulate a group of classes, sub packages and interfaces. Packages are used for: Preventing naming conflicts.

STEP 7

THREADS ,TOOLS

Higher priority threads are executed before lower priority threads. Thread is critical in the program because it enables multiple operations to take place within a single method.
